
# ☕ Jornada de Estudos em Java Backend

Este repositório é destinado a registrar meus estudos, projetos e evolução no desenvolvimento **backend com foco em Java**.

Atualmente, não possuo experiência prévia com a linguagem Java e tenho apenas conhecimentos básicos sobre **Programação Orientada a Objetos (POO)**. Por isso, esta jornada representa um grande desafio, mas também uma excelente oportunidade de aprendizado e crescimento profissional.

## 🎯 Objetivo

Meu principal objetivo é desenvolver as habilidades necessárias para conquistar uma **vaga de estágio na área de desenvolvimento de software**, construindo uma base sólida em:

- Java
- Programação Orientada a Objetos
- Desenvolvimento Backend
- Bancos de Dados
- APIs REST
- Frameworks do ecossistema Java
- Boas práticas de desenvolvimento

## 🚀 Sobre esta Jornada

Ao longo deste repositório, pretendo documentar:

- Resumos teóricos
- Exercícios práticos
- Projetos de estudo
- Desafios encontrados
- Soluções e aprendizados
- Evolução técnica ao longo do tempo

A ideia é manter um histórico da minha evolução, servindo tanto como material de consulta quanto como uma forma de acompanhar meu progresso.
